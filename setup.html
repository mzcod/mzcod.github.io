Setting up

Following the official quick start guide

It's quite simple, especially in kernels newer than 5.6 with WireGuard built-in.

Docker

Using docker images like linuxserver/wireguard to set up a server, and kizzx2/wireguard-socks-proxy can let you connect to WireGuard servers as a socks5 proxy. Performance would still be good, as LXC containers share the same kernel with the host.
